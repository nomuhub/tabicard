配布方法ガイド
  • このHTMLファイル自体をLINEやDiscordで送る → 受け取った人はブラウザで開くだけ、インストール不要
  • 全カードPNGをダウンロード → Googleドライブや共有フォルダにまとめて置く
  • URLで共有 → カード情報（テキスト）をリンク1本で送れる。受け取った人はインポートで再現できる
  • GitHub Pagesに置けば永続URLになる（無料・アカウント登録のみ）
